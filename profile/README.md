## 스터디 방식 📚
1. 매주 화요일 오전 10시 ~ 오후 1시, 충무로 근처에서 총 3시간 동안 진행한다. (1기 때는 주 2회 2시간씩 진행)
2. 2시간 동안 3~4문제를 푼 후, Github 각자 레포지토리에 Push한다.
3. 이후 1시간 동안 자신의 코드를 설명하고, 상호 피드백을 진행한다.
4. 문제는 백준에서 가져오며, 스터디장이 출제한다.
5. 사유 없는 지각 시, 분당 500원의 지각비를 걷는다 🤑
